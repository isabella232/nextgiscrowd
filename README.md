NextGIS Crowd
============

Crowdsourcing tool to collaboratively collect and edit spatial data.

## Official website

http://nextgis.github.io/nextgiscrowd

## Example projects using NGC

* [uikgeo](http://uikgeo.ru)

## Current limitations:

* points only
